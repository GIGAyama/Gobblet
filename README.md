# ゴブレット (Gobblet) on GAS

Google Apps Script (GAS) で動作する、3x3マスの戦略ボードゲーム「ゴブレット」風のWebアプリケーションです。

## 概要

このゲームは、2人対戦用のボードゲームです。自分の色の駒を、相手よりも先に「縦・横・斜め」のいずれか1列に3つ並べることを目指します。

### ゲームのルール
1. **駒のサイズ**: 各プレイヤーは「大・中・小」の3種類のサイズの駒を持っています。
2. **被せる**: 盤面上にある駒よりも「大きい」サイズの駒であれば、上から被せて隠すことができます。
3. **移動**: 自分の手番では、手持ちの駒を置くだけでなく、既に盤面に置いた自分の駒を別のマスへ移動させることもできます。
4. **勝利条件**: 先に3つ自分の駒を並べたプレイヤーの勝利です。

## 特徴

- **GASで完結**: サーバーサイド (`Code.gs`) とフロントエンド (`index.html`) のみで構成されており、複雑な環境構築なしで公開可能です。
- **温かみのあるデザイン**: 小学校の授業やレクリエーションでも使いやすいよう、「M PLUS Rounded 1c」フォントを使用した丸みのあるデザインを採用しています。
- **勝利演出**: 勝利時にはキャンバスアニメーションによる紙吹雪が舞い、対戦を盛り上げます。
- **ユニバーサルデザイン**: 低学年でも読めるよう、重要なテキストにはルビを付与しています。

## 使い方（インストール方法）

1. [Google Apps Script](https://script.google.com/) で新しいプロジェクトを作成します。
2. プロジェクト内に `Code.gs` と `index.html` という名前でファイルを作成し、本リポジトリのコードをそれぞれコピー＆ペーストします。
3. 画面右上の「デプロイ」ボタンから「新しいデプロイ」を選択します。
4. 種類を「ウェブアプリ」にし、アクセスできるユーザーを設定してデプロイを実行します。
5. 発行されたURLにアクセスするとゲームが開始されます。
