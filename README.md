# パクパクゴブレット

Webブラウザだけで動作する、3x3マスの戦略ボードゲーム「ゴブレット」風のアプリケーションです。

## 概要

このゲームは、1つの端末を使って2人で遊ぶ対戦用のボードゲームです。自分の色の駒を、相手よりも先に「縦・横・斜め」のいずれか1列に3つ並べることを目指します。

### ゲームのルール

1.  **駒のサイズ**: 各プレイヤーは「大・中・小」の3種類のサイズの駒をそれぞれ2つずつ持っています。
    
2.  **被せる**: 盤面上にある駒よりも「大きい」サイズの駒であれば、上から被せて隠すことができます（相手の駒にも自分の駒にも被せられます）。
    
3.  **移動**: 自分の手番では、手持ちの駒を置くだけでなく、既に盤面に置いた自分の駒を別のマスへ移動させることもできます。※移動させた結果、相手の駒が3つ揃ってしまった場合は負けになります。
    
4.  **勝利条件**: 先に3つ自分の駒を並べたプレイヤーの勝利です。
    

## 特徴

*   **シングルファイル構成**: サーバー側の処理（GASやPHPなど）は不要です。HTMLファイル1つで完全に動作するため、ダウンロードしてブラウザで開くだけですぐに遊べます。
    
*   **温かみのあるデザイン**: 小学校の授業やレクリエーションでも使いやすいよう、「Zen Maru Gothic」フォントを使用した丸みのあるデザインを採用しています。
    
*   **音と動きの演出**: SweetAlert2を用いたわかりやすいポップアップ表示や、Web Audio APIによる効果音で対戦を盛り上げます。
    
*   **ユニバーサルデザイン**: 低学年でも読めるよう、重要なテキストやルール説明にはルビ（ふりがな）を付与しています。
    

## 使い方（遊び方・公開方法）

### ローカルで遊ぶ場合

1.  本リポジトリの `index.html` をダウンロードします。
    
2.  ダウンロードしたファイルをダブルクリックして、Webブラウザ（Chrome, Edge, Safariなど）で開くだけでゲームが開始されます。
    

### Web上に公開する場合（GitHub Pagesなど）

1.  GitHub等のホスティングサービスでリポジトリを作成します。
    
2.  `index.html` を配置します。
    
3.  GitHub Pagesの機能などを有効化するだけで、全世界に公開可能です。
    

## 技術スタック

*   **フロントエンド**: HTML / CSS / Vanilla JavaScript
    
*   **UIフレームワーク**: Bootstrap 5
    
*   **アラート・モーダル**: SweetAlert2
    
*   **サウンド**: Web Audio API
    
*   **フォント**: Google Fonts (Zen Maru Gothic)
    
*   **アイコン/駒**: インラインSVG
